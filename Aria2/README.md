安装命令

bash <(curl -Lsk https://cdn.jsdelivr.net/gh/xzc08218/cdn/Aria2/ccaa.sh) cdn

下载路径（请填写绝对地址，默认/data/ccaaDown）

访问地址：http://你的ip地址:6080

File Browser 用户名:ccaa'

File Browser 密码:admin'

常用命令
安装成功后，您可以使用如下命令来管理CCAA：

#进入CCAA管理界面
ccaa
#查看ccaa状态
ccaa status
#启动ccaa
ccaa start
#停止ccaa
ccaa stop
#重启ccaa
ccaa restart
#查看当前版本
ccaa -v
